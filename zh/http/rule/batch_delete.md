# 批量删除策略

批量删除已经录入的策略

## 请求路径

> `/v2/rule`

## 请求方式

> DELETE

| 字段 | 类型         | 必填 | 字段释义           |
| ---- | ------------ | ---- | ------------------ |
| ids  | String array | N    | 需要批量删除策略id |

## 请求示例:

删除`rule_id`为102,103 的策略。

> `/v2/rule/`

```
{
    "ids": [
        "102",
        "103"
    ]
}
```

## 返回示例

```json
{
  "data": null,
  "code": 200,
  "msg": "OK"
}
```
