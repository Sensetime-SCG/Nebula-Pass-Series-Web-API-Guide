# 批量删除人员

批量删除已经录入的人员

## 请求路径

> `/v2/user`

## 请求方式

> DELETE

| 字段 | 类型         | 必填 | 字段释义           |
| ---- | ------------ | ---- | ------------------ |
| ids  | String array | N    | 需要批量删除人员id |

## 请求示例:

批量删除`user_id`为*3*，`user_id`为*4*的人员。

> `/v2/user`

```
{
    "ids": [
        "3",
        "4"
    ]
}
```

## 返回示例

```json
{
    "data": {
        "errors": []
    },
    "code": 200,
    "msg": "OK"
}
```
