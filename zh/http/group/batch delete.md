# 批量删除通行组

批量删除已经录入的通行组

## 请求路径

> `/v2/group`

## 请求方式

> DELETE

| 字段 | 类型         | 必填 | 字段释义             |
| ---- | ------------ | ---- | -------------------- |
| ids  | String array | N    | 需要批量删除通行组id |

## 请求示例:

删除`group_id`为1，`group_id`为2的通行组。

> `/v2/group`

```
{
    "ids": [
        "1",
        "2"
    ]
}
```

## 返回示例

```json
{
    "data": {
        "errors": []
    },
    "code": 200,
    "msg": "OK"
}
```
